# Curated Training Dataset

This folder is the staging ground for the "Golden Dataset" that will eventually be used to fine-tune the Tarsus AI model.

## The Artisanal Curation Workflow
As you manually review your Markdown files (e.g., ~10 files a day), copy the perfected, deeply theological files into this folder.

**Checklist before placing a file here:**
- [ ] Are all conversational "scratchpad" notes removed?
- [ ] Is the vocabulary consistent? (e.g., *innerstand* instead of *understand*, *cOLD* covenant)
- [ ] Is the formatting clean?
- [ ] Does it accurately represent the core teachings of Daniel Miles?

Once this folder has a critical mass of files (e.g., 100 to 300 highly curated studies), we can run the conversion script to format these specifically for GPU fine-tuning!
