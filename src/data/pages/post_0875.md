## Content

While EVENTS portrayed in media seem to suggest the whole world is mad, I REMIND myself I should think only the best OF everyone, to thereby bring forth only the best FOR everyone: Three BEST wishes: Grace→Mercy→Peace. 
Let's all put away the blame game of law, and politics, for it only results in shame on us all, if we all are one. One what? Let's all think on these things, noted in Philippians 4:8 KJV: 
- whatsoever things are true, 
- 
whatsoever things [are] honest, 
- whatsoever things [are] just, 
- whatsoever things [are] pure, 
- whatsoever things [are] lovely, 
- whatsoever things [are] of good report;
- if [there be] any virtue, and if [there be] any praise, 
think on these things.
