Add  Bible Search Tools  to your site!


You will find four (4) sections on this page. Each one corresponds to one type of search for the Bible Search Tools. Each section is divided into two parts: The functional search form and immediately below it the source code. Just copy the source code directly from the page to the appropriately formatted web page of your choice. You can change some things, such as formatting and descriptions, but you must not change any of the rest of the code or the Search will not work.

Or if you prefer, you can simply download the following zipfile. This zipfile contains a web page that already has all of the Bible Search Tools on it, with all of the images and formatting already done for you!
